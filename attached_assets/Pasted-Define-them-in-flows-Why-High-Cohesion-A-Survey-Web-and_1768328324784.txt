Define them in flows/.

Why? High Cohesion. A "Survey" (Web) and its "Result" (SMS) are two sides of the same coin. If you define the survey questions in flows/marketing_hooks.yaml alongside the SMS logic, you can edit the entire campaign in one file. If you separate them, you risk the survey sending data (e.g., "Result: Type_A") that the SMS flow doesn't know how to handle.

Here is the implementation plan:

The YAML: We add a web_survey block to your flow definitions.

The Python: We update the route to load that block dynamically: /assessment/<survey_id>.

The HTML: We rewrite assessment.html to be a generic rendering engine.

Step 1: The YAML Structure (flows/marketing_hooks.yaml)
We add a web_survey key. This defines the URL slug, title, and questions.

YAML

flows:
  # ----------------------------------------------------

  # URL: neuvero.ai/assessment/style
  # ----------------------------------------------------
  hook_style_quiz:
    description: "LinkedIn Hook - Communication Style"
    # This ID matches the URL slug
    web_survey:
      slug: "style"
      title: "Discover Your Neuro-Leadership Style"
      description: "Are you a Systemizer or an Empathizer? Take the 3-question diagnostic."
      questions:
        - id: "q1"
          text: "A project is failing. What is your first instinct?"
          options:
            - text: "Fix the Process (Data, Timelines)"
              score_key: "sys" 
            - text: "Fix the People (Morale, Alignment)"
              score_key: "emp"
        
        - id: "q2"
          text: "In a heated meeting, what annoys you more?"
          options:
            - text: "People being rude or blunt."
              score_key: "emp"
            - text: "People talking without data."
              score_key: "sys"
              
      results:
        # Simple majority wins logic (handled in JS)

          sys: "Systemizer"

    triggers: []
    steps:
      - type: response

Step 2: The Python Route (app.py)
Update your route to accept a slug. It searches your db.flows for a matching web_survey.slug.

Python

@app.route('/assessment/<slug>', methods=['GET'])
def show_dynamic_assessment(slug):
    # 1. Search for the flow that owns this survey slug
    survey_data = None
    
    # We iterate through all loaded flows to find the matching slug
    # (In a real DB, this would be a query. For memory, loop is fine.)
    for flow_id, flow_def in db.flows.items():
        web_config = flow_def.get('web_survey')
        if web_config and web_config.get('slug') == slug:
            survey_data = web_config
            break
    
    if not survey_data:
        return "Survey not found", 404

    # 2. Render the generic template with specific data
    return render_template('assessment_engine.html', survey=survey_data)

    <script>
        // Pass Python data to JS
        const totalQuestions = {{ survey.questions|length }};
        const profiles = {{ survey.results.profiles|tojson }};
        
        let scores = {};
        
        function nextSlide(currentIndex) {
            // Hide current
            const currentId = currentIndex === 0 ? 'slide-intro' : 'slide-' + (currentIndex - 1);
            document.getElementById(currentId).classList.remove('active');
            
            // Show next
            let nextId;
            if (currentIndex < totalQuestions) {
                nextId = 'slide-' + currentIndex;
            } else {
                nextId = 'slide-final';
            }
            document.getElementById(nextId).classList.add('active');
        }

        function recordAnswer(key) {
            if (!scores[key]) scores[key] = 0;
            scores[key]++;
        }

        function calculateResult() {
            // Simple Majority Wins
            let maxKey = null;
            let maxVal = -1;
            for (const [key, val] of Object.entries(scores)) {
                if (val > maxVal) {
                    maxVal = val;
                    maxKey = key;
                }
            }
            // Fallback if generic
            if (!maxKey && Object.keys(profiles).length > 0) {
                 maxKey = Object.keys(profiles)[0];
            }
            return profiles[maxKey] || "General Profile";
        }

        function submitQuiz() {
            const phone = document.getElementById('phone').value;
            const name = document.getElementById('first_name').value;
            const result = calculateResult();

            fetch('/hooks/typeform', {
                method: 'POST',
                headers: { 'Content-Type': 'application/json' },
                body: JSON.stringify({
                    phone: phone,
                    first_name: name,
                    calculated_profile: result
                })
            }).then(response => {
                if(response.ok) {
                    document.getElementById('displayPhone').innerText = phone;
                    document.getElementById('slide-final').classList.remove('active');
                    document.getElementById('slide-success').classList.add('active');
                } else {
                    alert("Error. Please check your number.");
                }
            });
        }
    </script>
